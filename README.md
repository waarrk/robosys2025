# kana-sort

![test](https://github.com/ryuichiueda/robosys2025/actions/workflows/test.yml/badge.svg)

標準入力から英語の文を読んで、間違っている疑いのあるものを出力します。

## とりあえず使ってみてください

```bash
$ git clone ...
$ cd robosys2025
$ echo aho | ./uspell
aho: こんな英語ない
```
